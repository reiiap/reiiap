# 💫 About Me

I'm an Indonesian software developer with a strong passion for building high-performance applications, game servers, and modern web platforms.

My primary expertise includes Minecraft server development, Java plugin engineering, professional SA-MP server systems, website development, Linux server administration, and backend infrastructure.

I enjoy solving complex technical challenges, optimizing performance, and designing scalable architectures that are reliable and maintainable.

Beyond my current work, I aspire to become a professional game developer capable of designing and developing complete games from scratch—from gameplay programming and networking to rendering, backend systems, and game architecture.

I believe continuous learning, clean code, and attention to detail are the foundation of exceptional software.
